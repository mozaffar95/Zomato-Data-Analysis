# Zomato Data Analysis

## Project Overview

This project analyzes Zomato restaurant data to understand ratings, cost and restaurant types.

## Dataset

The dataset includes:

* Restaurant Name
* Online Order
* Table Booking
* Rating
* Votes
* Approx Cost
* Restaurant Type

## Tools Used

* SQL
* Python
* CSV Dataset
* GitHub

## Analysis

In this project I performed:

* Data Cleaning
* Restaurant rating analysis
* Cost analysis
* Online order analysis
* Restaurant type analysis

## Conclusion

This project helps understand restaurant trends using Zomato dataset.
